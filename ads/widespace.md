# Widespace

## Examples

### Basic

```html
<amp-ad
  width="320"
  height="320"
  type="widespace"
  data-sid="f666bfaf-69cf-4ed9-9262-08247bb274e4"
>
</amp-ad>
```

### More parameters

```html
<amp-ad
  width="320"
  height="320"
  type="widespace"
  data-sid="f666bfaf-69cf-4ed9-9262-08247bb274e4"
  data-demo-Gender="F"
  data-demo-Country="SE"
  data-demo-Region="W1U 8EW"
  data-demo-City="Stockholm"
  data-demo-Yob="1981"
>
</amp-ad>
```

## Configuration

For configuration details, please contact integrations@widespace.com.

### Required parameters

- `data-sid`

### Optional demography parameters

- `data-demo-Gender`
- `data-demo-Country`
- `data-demo-Region`
- `data-demo-City`
- `data-demo-Yob`
