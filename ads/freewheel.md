# FreeWheel

## Example

### Expand-banner

```html
<amp-ad width="300" height="250"
  type="freewheel"
  data-zone="2003"
  layout="responsive">
</amp-ad>
```

### FloorAd

```html
<amp-sticky-ad layout=”nodisplay”>
  <amp-ad width="350" height="150"  
    type="freewheel" 
    data-zone="2003">
  </amp-ad>
</amp-sticky-ad>
```

## Configuration

For details on the configuration semantics, please contact the FreeWheel support team : clientsidesdk@freewheel.tv

Supported parameters:  
All parameters are optional, unless otherwise stated  
- `data-zone` [required]
- `data-blurDisplay`
- `data-timeline`
- `data-soundButton`
- `data-defaultMute`
- `data-onOver`
- `data-closeAction`
- `data-errorAction`
- `data-pauseRatio`
- `data-label`
- `data-vastUrlParams`
- `data-gdpr`
- `data-gdpr_consent`
- `data-gdpr_consented_providers`
- `data-useCMP`
