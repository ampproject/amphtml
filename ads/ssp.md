# SSP

## Example

```html
<amp-ad
  width="300"
  height="250"
  data-id="cnt-1234"
  data-width="300"
  data-height="250"
  data-zoneid="1234"
>
</amp-ad>
```

## Configuration

Required parameters:

- `data-id`
- `data-width`: Width of position (must be same as width attribute)
- `data-width`: Height of position (must be same as height attribute)
- `data-zoneid`: Id of zone
