# AJA

The AJA Ads for AMP supports Banner, Native, Video, and Recommendation Widget.
You have to contact our account manager to create ad spot at first.

## Examples

### Basic

```html
  <amp-ad width="300" height="250"
             type="aja"
             layout="responsive"
             data-asi="ZZZZZ">
  </amp-ad>
```

## Configuration

Required parameters:

- asi
  - `data-asi`
