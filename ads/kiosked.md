# Kiosked

## Example

```html
<amp-ad
    width="300"
    height="250"
    type="kiosked"
    data-scriptid="91">
</amp-ad>
```

## Configuration

You need to get a scriptid parameter from one of our account managers. 
For testing purposes you can use data-scriptid="91".

Required parameters:

- data-scriptid
