# OneAD

## Example
```html
  <amp-ad width="300" height="250"
      type="onead"
      layout="responsive"
      data-playmode="amp-inread"
      data-uid="1000033"
      data-pid="48123">
  </amp-ad>
```
## Configuration
Supported parameters:

- `data-uid`
- `data-playmode`
- `data-pid` (optional)

For configuration details, please contact [OneAD](https://www.onead.com.tw/)
