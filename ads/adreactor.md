# AdReactor

## Example

```html
<amp-ad width="728" height="90"
    type="adreactor"
    data-pid="790"
    data-zid="9"
    data-custom3="No Type">
</amp-ad>
```

## Configuration

For details on the configuration semantics, please contact the ad network or refer to their documentation. 

Supported parameters:

- `data-pid`
- `data-zid`
- `data-custom3`
