# Wisteria

## Example

```html
<amp-ad
  type="wisteria"
  width="300"
  height="480"
  layout="responsive"
  data-site-id="8"
  data-template-number="0"
>
</amp-ad>
```

### Configuration

For configuration details, please contact https://www.excite.co.jp/guide/wisteria/

Supported parameters:

- data-site-id (required)
- data-template-number (required)
- height (optional)
- width (optional)
