# Dable

## Example

### Basic

```html
<amp-embed width="500" height="200"
     type="dable"
     data-widget-id="jobgqR7W"
     data-item-id="testitem">
</amp-embed>
```

## Configuration

For configuration details and to generate your tags, please contact https://admin.dable.io.

### Required parameters

- `data-widget-id`

### Optional parameters

- `data-item-id`
- `data-service-name`
