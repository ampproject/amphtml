# Sulvo

## Example

### Regular ad

```html
<amp-ad
  width="300"
  height="250"
  type="sulvo"
  data-ad="example.com_display"
  class="demand-supply"
>
</amp-ad>
```

### Sticky ad

```html
<amp-sticky-ad>
  <amp-ad
    width="300"
    height="250"
    type="sulvo"
    data-ad="example.com_sticky_mobile"
    class="demand-supply"
  >
  </amp-ad>
</amp-sticky-ad>
```

## Configuration

For further information, please contact [Sulvo](https://sulvo.com/).

### Required parameters

-   `data-ad`: Ad ID provided by Sulvo.
