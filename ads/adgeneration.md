# Ad Generation

## Example

```html
<amp-ad width="300" height="250"
    type="adgeneration"
    data-id="10722">
</amp-ad>
```

## Configuration

For details on the configuration semantics, please see [Ad Generation's documentation](https://github.com/AdGeneration/sdk/wiki).

Supported parameters:

- `data-id`
- `data-targetid`
- `data-displayid`
- `data-adtype`
- `data-async`
- `data-option`
