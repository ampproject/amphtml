# WP Media (Wirtualna Polska)


## Examples

```html
<amp-ad height="320"
    type="wpmedia"
    data-bunch="7757"
    data-sn="sg"
    data-slot="3"
    data-slots="3,10">
</amp-ad>
```

```html
<amp-ad height="250"
    type="wpmedia"
    data-bunch="7757"
    data-sn="sg"
    data-slot="10"
    data-slots="3,10">
</amp-ad>
```


### Required parameter

- `data-slot`
- `data-bunch`

### Optional parameters

- `data-slots`
- `data-sn`
