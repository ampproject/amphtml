# Adman

## Example

```html
<amp-ad width="300" height="250"
    type="adman"
    data-ws="17342"
    data-s="300x250"
    data-host="talos.adman.gr">
  </amp-ad>
```

## Configuration

For semantics of configuration, please see ad network documentation.

Supported parameters:

- data-ws
- data-s
- data-host
