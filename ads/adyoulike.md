# Adyoulike

Serves ads from [Adyoulike](https://www.adyoulike.com/).

## Example

### Normal ad

```html
<amp-ad width="300" height="250"
      type="adyoulike"
      data-placement=""
</amp-ad>
```

## Configuration

For configuration details and to generate your tags,  please contact [Adyoulike](https://www.adyoulike.com/#contact).

### Required parameters

- `data-placement`

### Optional parameters

- `data-dc`
