# MADS

## Example

### Basic

```html
<amp-ad
  width="320"
  height="50"
  type="mads"
  data-adrequest='{"pid":"6252122059"}'
>
</amp-ad>
```

## Configuration

For semantics of configuration, please see [MADS documentation](http://wiki.mads.com/sites/javascript-ad-tags/).

Supported parameters:

- `data-adrequest`: MADS ad request parameters
