Before using this example page, please add the following to extensions/amp-analytics/0.1/vendors.js: 
      'example-3p-vendor': {
        'transport': {'beacon': true, 'xhrpost': true, 'image': true},
      },
Then look at the network log, and see the requests including "imgData="
