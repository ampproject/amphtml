## Behavior

The `amp-google-document-embed` component displays document files like Word documents,
Excel spreadsheets, and PDFs.

Example:
```html
<amp-google-document-embed
    src="https://www.example.com/document.pdf"
    width="800"
    height="600"
    layout="responsive">
</amp-google-document-embed>
```
