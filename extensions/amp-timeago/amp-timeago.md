## Behavior

The <code>amp-timeago</code> component is a wrapper around the <a href="https://github.com/hustcc/timeago.js">timeago.js</a> project for use on AMP pages.

## Attributes

**datetime** (required)

An ISO datetime. E.g. 2017-03-10T01:00:00Z.
