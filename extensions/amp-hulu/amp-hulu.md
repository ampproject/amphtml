## Example

```html
<amp-hulu width="412" height="213" layout="responsive"
  data-eid="4Dk5F2PYTtrgciuvloH3UA">
</amp-hulu>
```

## Attributes

**data-eid**

In a URL like https://secure.hulu.com/embed.html?eid=4Dk5F2PYTtrgciuvloH3UA `4Dk5F2PYTtrgciuvloH3UA` is the eid.
