amp-brightcove examples

Ads

Any Brightcove Player in AMP can play ads. Player 6.18.1 makes the configuraiton easier with builtin mapping of query parameters to global variables.

The same player is used in each example below, the only difference being the value of a data-param-ad-id on each embed.

The ad configuration url uses macro {pageVariable.adId}. The player configuraiton has {"query_string_to_window": {"globals": ["adId"]}}.

Preroll and bumper

The amp-brightcove element has data-param-ad-id="preonlybumper"

Postroll

The amp-brightcove element has data-param-ad-id="postonly"
