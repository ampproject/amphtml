The amp-geo config on this page is: 
    
    <amp-geo layout="nodisplay">
      <script type="application/json">
        {
          "ISOCountryGroups": {
            "nafta": [ "ca", "mx", "us", "unknown" ],
            "unknown": [ "unknown" ],
            "anz": [ "au", "nz" ]
          }
        }
      </script>
    </amp-geo>

This means that if your country is "unknown" you will appear in the "nafta" and "unknown" groups. To change the country for testing append #amp-go=<code> to the url and reload the page (eg #amp-geo=nz)
